# Pix Security API

API REST desenvolvida com Spring Boot para gerenciamento de clientes.

## Funcionalidades

- Cadastro de clientes
- Atualização de dados
- Inativação de clientes
- Validação de dados
- Tratamento global de erros
- Documentação com Swagger

## Tecnologias

- Java 21
- Spring Boot
- Spring Data JPA
- Bean Validation
- MySQL / H2
- Swagger (OpenAPI)

## Endpoints principais

- POST /api/v1/customers
- PUT /api/v1/customers/{id}
- PATCH /api/v1/customers/{id}/deactivate

## Objetivo

Projeto desenvolvido para prática de backend com foco em boas práticas de API REST.
